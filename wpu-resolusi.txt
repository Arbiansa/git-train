Python
Python adalah bahasa pemrograman tingkat tinggi yang populer karena sintaksisnya yang sederhana dan kemampuannya yang kuat. Python sering digunakan untuk pengembangan web, data science, machine learning, otomatisasi, dan berbagai bidang lainnya. Python mendukung berbagai paradigma pemrograman seperti pemrograman berorientasi objek (OOP) dan pemrograman fungsional.

List di Python
List adalah salah satu tipe data bawaan di Python yang digunakan untuk menyimpan kumpulan elemen dalam satu variabel. List bersifat mutable (dapat diubah setelah dibuat), dan elemen-elemennya dapat berupa tipe data apa saja, bahkan campuran dari berbagai tipe data.

Contoh List:
=======================================================
script

# Membuat list
my_list = [1, "hello", 3.14, True]

# Mengakses elemen list
print(my_list[1])  # Output: hello

# Menambahkan elemen ke dalam list
my_list.append(10)

# Menghapus elemen dari list
my_list.remove("hello")
Fitur Utama:
=======================================================

Dapat berisi berbagai tipe data.
Mendukung operasi penambahan, penghapusan, dan pengubahan elemen.
Dapat diakses menggunakan indeks (mulai dari 0).
